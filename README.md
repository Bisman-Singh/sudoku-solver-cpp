# Sudoku Solver

A 9x9 Sudoku puzzle solver using backtracking, with an interactive menu and built-in puzzles of varying difficulty.

## Features

- Solves any valid 9x9 Sudoku puzzle using backtracking
- Pretty-printed board with grid lines
- 3 built-in puzzles: easy, medium, and hard
- Custom puzzle input support
- Input validation with conflict detection
- Solving time measurement
- Interactive menu for puzzle selection
- Option to solve all built-in puzzles at once

## Build

```bash
make
```

## Usage

```bash
./sudoku
```

Select a puzzle from the menu or enter your own. Use `0` to represent empty cells when entering a custom puzzle.
